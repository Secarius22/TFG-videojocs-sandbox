# Ashen of Realms

## Descripción
Ashen of Realms es un videojuego 2D de tipo sandbox con estética pixel art y ambientación de fantasía medieval. El objetivo es crear un mundo explorable con generación procedural y mecánicas básicas de interacción.

## Estado actual
Este repositorio contiene una versión inicial del proyecto, incluyendo:
- Primeros assets gráficos (personaje, tiles)
- Configuración del proyecto en Unity
- Primeras pruebas de animación

## Tecnologías utilizadas
- Unity (motor de desarrollo)
- Aseprite (creación de sprites)
- GitHub (control de versiones)

## Ejecución
Para ejecutar el proyecto:
1. Clonar el repositorio
2. Abrir con Unity Hub
3. Seleccionar la escena principal
